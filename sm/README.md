# Scripture Matrix

Matrix-style falling text animations featuring scriptures from various books.

## Available Versions

| File | Scripture Source |
|------|------------------|
| `bible-matrix.html` | King James Bible |
| `bom-matrix.html` | Book of Mormon |
| `dnc-matrix.html` | Doctrine and Covenants |
| `pogp-matrix.html` | Pearl of Great Price |
| `scripture-matrix.html` | Combined scriptures |

## Features

- **Matrix Rain Effect**: Classic green falling characters
- **Scripture Text**: Random scripture verses cascade down
- **Full Screen**: Canvas fills the viewport
- **Responsive**: Adapts to window size
- **Multiple Sources**: Choose your preferred scripture

## Usage

1. Open any of the HTML files in a web browser
2. Watch scriptures fall in Matrix style
3. Refresh for different random verses

## Technical Details

- HTML5 Canvas API
- RequestAnimationFrame animation
- Scripture data embedded or loaded from JSON
- Vanilla JavaScript
- No dependencies

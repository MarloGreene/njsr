# Network Information (IP2)

Display detailed network information including IPv4 and IPv6 addresses.

## Features

- **IP Address Display**: Shows your public IP addresses
- **IPv4 & IPv6**: Both address types when available
- **Network Details**: Additional connection information
- **Clean Interface**: Easy-to-read presentation

## Information Displayed

- Public IPv4 address
- Public IPv6 address (if available)
- Connection type
- Network provider information

## Usage

1. Open `index.html` in any web browser
2. View your network information
3. Information updates automatically

## Related Tools

- `/ip` - IP Information Dashboard (with browser fingerprinting)

## Technical Details

- Uses third-party IP lookup services
- Vanilla JavaScript
- Responsive design

## Privacy Note

This tool contacts external services to determine your public IP address. Your IP address is inherently visible to any website you visit.

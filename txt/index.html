Adding a New Corpus

  1. Create a new directory under /txt/ (e.g., /txt/shakespeare/)
  2. Add a corpus.config.json with title, author, and chunking rules
  3. Add your text files to a texts/ subdirectory
  4. Run the preprocessor to generate the search index:
node txt/engine/preprocess.js

See the /txt/sample/ corpus for a working example.

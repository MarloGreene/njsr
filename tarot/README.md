# Tarot Card Reader

A mystical tarot card reading application for guidance and reflection.

## Features

### Spread Options
- **Single Card**: Quick one-card draw
- **Past/Present/Future**: Classic 3-card spread
- **Celtic Cross**: Traditional 5-card spread

### Card Features
- **Full Major Arcana**: All 22 major arcana cards
- **Card Meanings**: Interpretations for each card
- **Upright/Reversed**: Cards can appear in either orientation
- **Visual Cards**: Beautiful card face designs

### Additional Features
- **Reading History**: Track past readings
- **Clear History**: Remove saved readings
- **Shuffle Animation**: Visual deck shuffling

## Spreads Explained

### Single Card
Draw one card for a quick insight or daily guidance.

### Past/Present/Future (3 cards)
- **Position 1**: Past - influences from your history
- **Position 2**: Present - current situation
- **Position 3**: Future - potential outcome

### Celtic Cross (5 cards)
A more detailed spread for complex questions.

## Usage

1. Open `index.html` in any web browser
2. Select a spread type (1, 3, or 5 cards)
3. Click "Draw Cards"
4. Click each card to reveal it
5. Read the interpretations
6. View history for past readings

## Files

- `index.html` - Main interface
- `tarot.css` - Styling and card designs
- `tarot.js` - Reading logic and card data

## Technical Details

- Card data embedded in JavaScript
- Random selection with Fisher-Yates shuffle
- localStorage for history
- CSS animations for card reveals
- Vanilla JavaScript
- No dependencies

## Privacy

- Reading history in localStorage
- No external connections
- No tracking

## Disclaimer

This is for entertainment and reflection purposes only. Tarot readings should not be used for making important life decisions.

# URL Tools

**STATUS: PAUSED - Client-side only**

URL encoding, decoding, and utility functions.

## Planned Features

- **URL Encoder**: Encode special characters for URLs
- **URL Decoder**: Decode percent-encoded URLs
- **Query Parser**: Parse and display URL query parameters
- **URL Builder**: Construct URLs from components

## Current Status

This tool is currently paused/in development. The client-side only implementation is available for basic encoding/decoding functions.

## Usage

1. Open `index.html` in any web browser
2. Use available encoding/decoding functions

## Technical Details

- Client-side JavaScript only
- Uses built-in `encodeURIComponent`/`decodeURIComponent`
- No server-side functionality

## Privacy

- All processing done locally
- No data sent to servers
- No tracking

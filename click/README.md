# Metronome (Click)

An interactive musical metronome with visual and audio feedback.

## Features

- **BPM Control**: Slider from 40-240 BPM
- **Visual Beat**: Visual indicator pulses with each beat
- **Preset Tempos**:
  - Largo (60 BPM)
  - Moderato (120 BPM)
  - Dubstep (140 BPM)
  - Presto (180 BPM)
- **Start/Stop**: Single button to control playback
- **Clean Interface**: Minimalist design for focus

## Usage

1. Open `index.html` in any web browser
2. Set desired BPM using the slider or preset buttons
3. Click "Start" to begin the metronome
4. Click "Stop" to pause

## Files

- `index.html` - Main structure
- `metronome.css` - Styling
- `metronome.js` - Audio and timing logic

## Technical Details

- Web Audio API for precise timing
- Vanilla JavaScript
- No dependencies
- Works in all modern browsers

## Privacy

- No data collection
- No localStorage usage
- Runs entirely in your browser

# Scripture Search (SS)

A fast scripture search variant for LDS standard works.

## Features

- **Quick Search**: Type to search scriptures instantly
- **Full Coverage**: All standard works included
- **Clean Interface**: Simple, focused design
- **Context Display**: See verses with surrounding text

## Scripture Coverage

- Holy Bible (King James Version)
- Book of Mormon
- Doctrine and Covenants
- Pearl of Great Price

## Usage

1. Open `index.html` in any web browser
2. Enter search terms
3. View matching verses
4. Click results for full context

## Related Tools

- `/lfss` - Lightning Fast Scripture Search (similar functionality)
- `/ssah` - Scripture Search & Highlighter (with annotation features)
- `/sh` - Scripture Highlighter (reading-focused)

## Technical Details

- Pre-loaded scripture data (~6.8MB)
- Client-side search
- Vanilla JavaScript
- No server required

## Privacy

- All data stored locally
- No external connections
- No tracking
- Works offline after load

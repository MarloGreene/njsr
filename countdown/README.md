# Countdown Timer

A customizable countdown timer for tracking time until important events.

## Features

- **Custom Events**: Set any event name
- **Date & Time Selection**: Pick precise target date and time
- **Live Countdown**: Real-time display of days, hours, minutes, seconds
- **Completion Alert**: Visual celebration when time reaches zero
- **Persistent**: Saves your countdown to localStorage

## Usage

1. Open `index.html` in any web browser
2. Enter an event name (e.g., "New Year 2027")
3. Select the target date and time
4. Click "Set Countdown"
5. Watch the countdown update in real-time

## Display Format

- **Days**: Total days remaining
- **Hours**: 00-23
- **Minutes**: 00-59
- **Seconds**: 00-59

## Files

- `index.html` - Main structure
- `countdown.css` - Styling
- `countdown.js` - Timer logic

## Technical Details

- Real-time updates using `setInterval`
- localStorage persistence
- Vanilla JavaScript
- No dependencies

## Privacy

- Countdown saved to localStorage
- No external connections
- No tracking

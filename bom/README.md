# Book of Mormon Reader

A web-based scripture reader for The Book of Mormon.

## Features

- **Complete Text**: Full Book of Mormon text
- **Chapter Navigation**: Easy navigation between books and chapters
- **Clean Reading Experience**: Distraction-free interface
- **Search Capability**: Find verses by keyword
- **Responsive Design**: Works on desktop and mobile

## Usage

1. Open `index.html` in any web browser
2. Navigate using the book/chapter selectors
3. Read and study the scriptures

## Data

Scripture data is stored in JSON format for efficient loading and navigation.

## Technical Details

- Vanilla JavaScript
- JSON scripture data
- No external dependencies
- Responsive CSS design

## Privacy

- All data stored locally
- No tracking or analytics
- Works offline after initial load

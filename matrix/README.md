# Matrix Rain

The iconic falling character animation from The Matrix, rendered on HTML canvas.

## Features

- **Classic Effect**: Green characters falling in columns
- **Pause/Resume**: Toggle animation on/off
- **Speed Control**: Adjust fall speed with slider
- **Full Screen**: Canvas fills the entire viewport
- **Responsive**: Adapts to window resize

## Usage

1. Open `index.html` in any web browser
2. Watch the Matrix rain effect
3. Click "Pause" to freeze the animation
4. Adjust speed with the slider

## Controls

- **Toggle Button**: Start/Stop animation
- **Speed Slider**: 10 (slow) to 100 (fast)

## Files

- `index.html` - Main structure
- `matrix.css` - Styling
- `matrix.js` - Canvas animation logic

## Technical Details

- HTML5 Canvas API
- RequestAnimationFrame for smooth animation
- Random character generation
- Vanilla JavaScript
- No dependencies

## Privacy

- No data collection
- No localStorage usage
- Runs entirely in your browser

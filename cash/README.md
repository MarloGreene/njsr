# Expense Tracker (Cash)

A personal expense tracking application to manage your budget and analyze spending habits.

## Features

### Expense Management
- **Add Expenses**: Description, amount, category, and date
- **Categories**: Food, Transportation, Shopping, Entertainment, Bills, Health, Education, Travel, Other
- **Edit/Delete**: Modify or remove expenses anytime

### Budget Tracking
- **Monthly Budget**: Set a spending limit
- **Remaining Balance**: Real-time tracking of budget vs. spending
- **Status Indicators**: Visual feedback on budget health

### Analysis
- **Total Expenses**: Current month spending summary
- **Category Breakdown**: Visual spending by category
- **Filtering**: By category, date range, or search term
- **Time Periods**: Today, This Week, This Month, This Year, All Time

### Data Management
- **Search**: Find expenses by description
- **Export CSV**: Download data for spreadsheets
- **Auto-save**: All data persisted to localStorage

## Usage

1. Open `index.html` in any web browser
2. Add expenses using the form at the top
3. Set a monthly budget using the "Set Budget" button
4. View and filter expenses in the list below
5. Check category breakdown for spending insights

## Files

- `index.html` - Main structure
- `styles.css` - Styling
- `script.js` - Application logic

## Keyboard Shortcuts

- Standard browser shortcuts for forms

## Privacy

- **100% Local**: All data stored in browser localStorage
- **No Server**: Nothing sent externally
- **No Tracking**: Zero analytics
- **Export Anytime**: Full control of your data

## Technical Details

- Vanilla JavaScript
- localStorage for persistence
- CSS Grid/Flexbox layout
- Works offline after initial load

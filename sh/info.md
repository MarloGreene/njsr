# Legacy Project Notes

This file contains early brainstorming and planning notes for the LDS Scripture Highlighter project.

**For up-to-date documentation, see `README.md` and `ROADMAP.md`.**

---

* Vanilla .html/.css/.js for privacy and localStorage
* Loads `lds-scriptures.txt` (one verse per line)
* Two-column book view, minimal chapter/verse markings
* Default highlight palette (green, yellow, pink, purple), fully configurable
* Click to highlight/cycle, drag to select
* Export/import highlight sets (future: markdown/html, smart sharing)
* Planned: configurable highlight sets, unique URLs, prebuilt sets (Scripture Mastery, Gospel Principles, etc.)

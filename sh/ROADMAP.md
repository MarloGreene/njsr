# Roadmap

This document outlines planned features and enhancements for the LDS Scripture Highlighter project.

## Planned Features

### Export/Import Functionality
- Export highlight sets as markdown or HTML
- Import highlight sets from exported files
- Preserve highlight data across devices/sessions

### Configurable Highlight Sets
- Create custom highlight sets with unique URLs for sharing
- Load highlight sets via URL parameters
- Community-shared highlight sets

### Pre-built Highlight Sets
- **LDS Scripture Mastery**: Highlight key scripture mastery verses
- **Gospel Principles**: Mark references to gospel principles
- **Divine Names**: Highlight all mentions of the Lord's names
- **Words of Christ**: Display Christ's words in red (like traditional Bibles)
- **Words of Mormon**: Highlight sections written by Mormon
- **Book of Mormon Witnesses**: Mark testimony verses
- **Temple References**: Highlight temple-related content
- **Atonement References**: Mark atonement-related scriptures

### Advanced Features
- Search functionality within scriptures
- Bookmark important verses
- Notes/annotation system for verses
- Print-friendly export with highlights
- Dark mode theme
- Font size and style customization

### Technical Improvements
- Performance optimization for large scripture files
- Progressive Web App (PWA) support for offline use
- Better mobile touch interactions
- Keyboard shortcuts for power users
- Accessibility improvements (ARIA labels, keyboard navigation)

## Implementation Priority

1. **High Priority**
   - Export/import highlight sets
   - Basic search functionality
   - PWA support

2. **Medium Priority**
   - Pre-built highlight sets
   - Notes/annotation system
   - URL-based highlight set sharing

3. **Low Priority**
   - Advanced themes
   - Print functionality
   - Additional customization options

# Lightning Fast Scripture Search (LFSS)

A high-performance scripture search engine for LDS scriptures.

## Features

- **Instant Search**: Results appear as you type
- **Full Text Search**: Search across all standard works
- **Verse Display**: See matching verses with context
- **Fast Performance**: Optimized for quick results
- **Keyboard Navigation**: Navigate results with keyboard

## Scripture Coverage

- Holy Bible (King James Version)
- Book of Mormon
- Doctrine and Covenants
- Pearl of Great Price

## Usage

1. Open `index.html` in any web browser
2. Start typing your search query
3. Results appear instantly
4. Click any result to view full verse

## Technical Details

- Pre-indexed scripture data
- Client-side search (no server needed)
- Optimized search algorithms
- Vanilla JavaScript
- Large data files (~6.8MB)

## Performance

The "Lightning Fast" name comes from:
- Pre-processed search index
- Efficient string matching
- Debounced input handling
- Minimal DOM updates

## Privacy

- All data stored locally
- No external connections
- No tracking or analytics
- Works offline after initial load

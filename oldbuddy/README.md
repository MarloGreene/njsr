# Buddy Letter Assistant

A comprehensive tool to help create supporting evidence letters (buddy letters) for VA PTSD disability claims.

## Purpose

Buddy letters are crucial supporting documents for VA disability claims. This tool helps veterans and their supporters create properly structured, effective buddy letters by:

- Organizing relevant information systematically
- Ensuring all important details are captured
- Following VA-recommended formats

## Features

### Information Capture
- **Relationship Selection**: Service member, family, friend, medical provider, etc.
- **Service Information**: Branch, unit, dates of service
- **Deployment Locations**: Iraq, Afghanistan, Kuwait, Qatar, and others
- **Combat Exposures**: IED attacks, direct combat, convoy operations, etc.

### Symptom Documentation
- **Re-experiencing Symptoms**: Nightmares, flashbacks, intrusive thoughts
- **Avoidance Symptoms**: Emotional numbness, avoiding reminders
- **Hyperarousal Symptoms**: Hypervigilance, startle response, sleep problems
- **Mood/Cognitive Symptoms**: Negative beliefs, guilt, emotional changes
- **Behavioral Changes**: Social withdrawal, relationship problems

### Timeline & Impact
- Condition before deployment
- When symptoms started
- Duration and progression
- Impact on daily life and functioning

### Data Management
- **Auto-save**: Progress saved to browser localStorage
- **Export**: Download as TXT or JSON
- **Import**: Restore from JSON backup

## Usage

1. Open `index.html` in any web browser
2. Complete each section of the form
3. Click "Continue to Letter" to generate the buddy letter
4. Review and customize the generated letter
5. Export for submission to the VA

## Privacy & Security

- **100% Local**: All data stored only in your browser
- **No Server**: Nothing sent to any external server
- **No Tracking**: Zero analytics or telemetry
- **Your Control**: Export and delete your data anytime

## Important Note

This tool helps organize information for buddy letters but does not constitute legal advice. For best results:
- Work with an accredited Veterans Service Organization (VSO)
- Be truthful and specific in all statements
- Include only information you personally witnessed or experienced

## Files

- `styles.css` - Styling

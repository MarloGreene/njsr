# PTSD Symptom Awareness Tool

An educational tool to help veterans understand PTSD symptoms and VA disability ratings.

## Purpose

This tool helps veterans:
- Understand common PTSD symptoms based on DSM-5 criteria
- See how combat exposures relate to potential claims
- Estimate VA disability compensation based on severity
- Connect with resources like VSOs and the VA

## Features

### Combat Exposure Tracking
- Combat MOS/Combat Arms
- Deployment to combat zone
- Outside the wire operations
- IDF/rocket/mortar attacks
- IED/VBIED exposure
- Direct combat engagement
- Witnessing death/injury
- Loss of battle buddies
- Convoy operations
- Handling human remains
- Military Sexual Trauma (MST)
- Multiple deployments

### Symptom Categories (DSM-5)
- **Intrusion**: Nightmares, flashbacks, intrusive memories
- **Avoidance**: Avoiding reminders, emotional numbness
- **Hyperarousal**: Hypervigilance, startle response, sleep issues
- **Mood/Cognitive**: Negative beliefs, guilt, emotional changes

### Compensation Calculator
- Severity slider (0-100%)
- Dependents calculator (spouse + children)
- 2026 VA compensation rates
- Real-time updates

### Resources
- Direct links to VA resources
- VSO finder
- VA hotline number

## Usage

1. Open `index.html` in any web browser
2. Check applicable combat exposures
3. Adjust severity slider based on symptom impact
4. Add dependents if applicable
5. See estimated monthly compensation
6. Review symptoms that may apply at your severity level

## Security & Privacy

- **Content Security Policy**: Strict CSP headers
- **No Referrer**: Prevents information leakage
- **100% Local**: No data sent anywhere
- **No Tracking**: Zero analytics or telemetry
- **Frame Protection**: Cannot be embedded in other sites

## Important Disclaimer

This tool is for **educational purposes only**. It does not:
- Diagnose any medical condition
- Guarantee any VA rating
- Replace professional medical or legal advice

Always work with:
- A licensed mental health professional
- An accredited Veterans Service Organization (VSO)
- The VA directly

# Background Gradient Generator

A visual CSS gradient generator with live preview.

## Features

- **Multiple Gradient Types**: Linear, Radial, and Conic gradients
- **Color Pickers**: Two color inputs for gradient endpoints
- **Angle Control**: Slider for precise angle adjustment (0-360 degrees)
- **Stripe Option**: Generate striped gradient patterns
- **Live Preview**: See changes instantly in the preview area

## Usage

1. Open `index.html` in any web browser
2. Select gradient type (Linear, Radial, or Conic)
3. Choose two colors using the color pickers
4. Adjust the angle with the slider
5. Toggle stripes option if desired
6. Copy the CSS for your project

## Files

- `index.html` - Main structure
- `style.css` - Styling
- `script.js` - Gradient generation logic

## Technical Details

- Vanilla JavaScript
- No dependencies
- Real-time CSS generation
- Works in all modern browsers

## Privacy

- No data collection
- No localStorage usage
- Runs entirely in your browser

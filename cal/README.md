# Calendar

A simple, clean yearly calendar display.

## Features

- **Full Year View**: All 12 months displayed at once
- **Current Date Highlight**: Today's date clearly marked
- **Clean Design**: Minimalist interface
- **Responsive Layout**: Grid adapts to screen size

## Usage

1. Open `index.html` in any web browser
2. View the current year's calendar
3. Current date is highlighted automatically

## Files

- `index.html` - Main structure
- `cal.css` - Styling
- `cal.js` - Calendar generation logic

## Technical Details

- Vanilla JavaScript
- CSS Grid layout
- No dependencies
- Auto-updates to current year

## Privacy

- No data collection
- No localStorage usage
- Runs entirely in your browser

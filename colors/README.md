# Color Palette Generator

Generate beautiful, random color palettes for your design projects.

## Features

### Palette Generation
- **5-Color Palettes**: Each generation creates 5 harmonious colors
- **Lock Colors**: Keep colors you like while regenerating others
- **Random Generation**: Press Space or click Generate for new colors

### Color Wheel
- **Visual Reference**: Built-in color wheel popup
- **Complementary Colors**: See opposite (complementary) colors at a glance

### Saving & History
- **Save Palettes**: Keep your favorite combinations
- **View Saved**: Browse all saved palettes
- **Copy HEX**: Click any hex code to copy to clipboard

### Keyboard Shortcuts
- **Space**: Generate new palette
- **Click color**: Lock/unlock that color
- **Click HEX**: Copy to clipboard

## Usage

1. Open `index.html` in any web browser
2. Press Space or click "Generate" for new colors
3. Click a color to lock it
4. Click "Save Palette" to keep a combination
5. Click "Saved" to view your collection
6. Click any HEX code to copy it

## Files

- `index.html` - Main structure
- `palette.css` - Styling
- `palette.js` - Palette generation logic

## Technical Details

- HSL-based color generation for pleasing palettes
- localStorage for saved palettes
- Canvas API for color wheel
- Vanilla JavaScript
- No dependencies

## Privacy

- Saved palettes stored in localStorage
- No external connections
- No tracking
